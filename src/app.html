Land Change Projections for Hawai'i

Change Accross the Landscape

The map below shows the change in land-use or carbon over the simulation period for the selected combination of scenario, iteration, and strata. Use the controls below the map to change the year, scenario, iteration or land-use layer on the map. The chart below displays trajectories of landscape change through time. Use the play, pause and stop controls or move the slider to animate the map through time.

Map Viewer

Layer controls

Use this interactive web map to explore land use and carbon data accross the landscape. Use the menu below the map frame on the left to update the model layers visible on the web map. These controls allow the user to toggle between different years, scenarios, iterations, and map products. Toggle the "Show map legend" button to show the legend for the current layer on the map. Frame controls

Use the plus and minus buttons on the top left of the web map to add or remove a map frame. When a new map frame is added, layer controls for the frame appear below the frame. Adding a frame allows the user to compare two or more layers side by side.

Base maps

Within each map frame hover over the layers icon to expand the basemap options. Grey-scale and imagery base maps are available as well as strata layers.

Animate map layers

Map layers may be animated through time using the controls below the graph. Choose the layers you would like to animate and then click the play button under the graph. Layers will update every five year timestep within the map frame. The graph tooltip will also animate within the graph frame and show land-use values.
